# Alan App Website

## Project Overview

The Alan App Website is a full-stack web application designed to provide an all-in-one allergy information platform. This project was deployed on Render.com and offers a user-friendly interface, robust backend, and smooth interaction between the frontend and backend.

## Features

- Built with React.js for a dynamic and responsive frontend.
- Node.js and Express.js backend with MongoDB for data persistence.
- REST API implementation for smooth interaction between the frontend and backend.
- Functionalities include creating, reading, updating, and deleting user reviews.

## Technologies Used

- **Frontend**:
  - React.js
  - React Router
  - CSS
- **Backend**:
  - Node.js
  - Express.js
  - MongoDB
- **Additional Features**:
  - Deployed on Render for seamless online accessibility.

## Project Structure

- **Frontend Folder**:
  - Contains all React components, pages, and static assets.
  - Example files:
    - `App.js`: Main React component.
    - `Navbar.js`: Component for navigation.
    - `Cards.js`: Component for displaying cards.
- **Backend Folder**:
  - Contains server logic, routes, and configuration files.
  - Example files:
    - `index.js`: Main server entry point.
    - `routes/`: Folder for API route definitions.
    - `models/`: MongoDB schema definitions.

## Deployment Link

Access the live application here: [Alan App Website](https://alan-app-website.onrender.com/)

## Running the Application

#### Install dependencies for both frontend and backend
```bash
npm install
```
#### Run Frontend
```bash
cd frontend
npm start
```
#### Run Backend
```bash
cd backend
node index.js
```
#### Access the application locally at:
```bash
http://localhost:3000
```
